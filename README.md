# Rails + Vue + Vite + TailwindCSS Template App

## Setup

Run the following commands to get started:

```bash
bundle install
rails db:create
rails db:migrate
```

To start the app, run:

```bash
bin/dev
```

This will start the Rails server and the Vite server concurrently.
